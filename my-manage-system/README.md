# my-manage-system

管理系统，练习使用 `vue-cli3` + `nodejs` + `mongodb` 搭建完整的项目工程。

## 常用命令

```js
yarn install   //Project setup
yarn run serve //启动开发模式
yarn run start //启动node服务

//Compiles and minifies for production
yarn run build

//Run your tests
yarn run test

//Lints and fixes files
yarn run lint
```
